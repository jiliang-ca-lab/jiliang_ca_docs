顶层设计与约束和点灯实验保持一致

```
module alu(
    input  [31:0] dip_sw,   // 输入开关
    output [15:0] leds      // 输出 LED 显示结果
);
//在这里实现你的逻辑
endmodule
```

**对dip_sw开关定义**

- 最高 8 位 `[31:24]` = **操作码**
- `[23:16]` 保留不用
- `[15:8]` = **alu_src1**（源操作数1）
- `[7:0]` = **alu_src2**（源操作数2）

**对leds定义**

- 高八位显示操作码，若操作码无效，则点亮最高位
- 低八位显示运算结果

**op定义如下：**

| 操作码 `op[7:0]` | 指令名称 | 功能       | 描述          |
| ---------------- | -------- | ---------- | ------------- |
| `8'h00`          | ADD      | A + B      | 加法          |
| `8'h01`          | SUB      | A - B      | 减法          |
| `8'h02`          | AND      | A & B      | 按位与        |
| `8'h03`          | OR       | A \| B     | 按位或        |
| `8'h04`          | XOR      | A ^ B      | 按位异或      |
| `8'h05`          | NOT      | ~A         | 按位取反      |
| `8'h06`          | SLL      | A << B     | 逻辑左移 B 位 |
| `8'h07`          | SRL      | A >> B     | 逻辑右移 B 位 |
| `8'h08`          | SRA      | A >>> B    | 算术右移 B 位 |
| `8'h09`          | ROL      | A 循环左移 | 循环左移 B 位 |
