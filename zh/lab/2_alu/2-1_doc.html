实验----设计中

Paul2026/4/4小于 1 分钟

顶层设计与约束和点灯实验保持一致

module alu(
    input  [31:0] dip_sw,   // 输入开关
    output [15:0] leds      // 输出 LED 显示结果
);
//在这里实现你的逻辑
endmodule

对dip_sw开关定义

  • 最高 8 位 [31:24] = 操作码
  • [23:16] 保留不用
  • [15:8] = alu_src1（源操作数1）
  • [7:0] = alu_src2（源操作数2）

对leds定义

  • 高八位显示操作码，若操作码无效，则点亮最高位
  • 低八位显示运算结果

op定义如下：

操作码 op[7:0]指令名称功能描述
8'h00ADDA + B加法
8'h01SUBA - B减法
8'h02ANDA & B按位与
8'h03ORA | B按位或
8'h04XORA ^ B按位异或
8'h05NOT~A按位取反
8'h06SLLA << B逻辑左移 B 位
8'h07SRLA >> B逻辑右移 B 位
8'h08SRAA >>> B算术右移 B 位
8'h09ROLA 循环左移循环左移 B 位
